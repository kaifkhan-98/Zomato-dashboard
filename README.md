# Zomato-dashboard
1. Project Title / Headline
Zomato Sales & Restaurant Analytics Dashboard | Power BI

2. Short Description
Developed an interactive Power BI dashboard to analyze Zomato restaurant and food ordering data. The dashboard provides insights into sales performance, customer behavior, restaurant ratings, food preferences, and year-over-year business growth to support data-driven decision-making.

3. Tech Stack
Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization
Excel/CSV Data Sources

4. Data Source
The dashboard uses Zomato-related datasets containing:
Restaurant information
Order transactions
Customer details
Food menu data
Ratings and reviews
Tables Used:
Orders
Restaurant
Users
Menu
Food
